# Calculadoras en python

Este repositorio contiene una colección de herramientas de cálculo desarrolladas en **Python**.
Es parte de mi aprendizaje de lógica de programación.
Este programa fue pensado para calcular y automatizar cosas que una calculadora normal no puede hacer
## proyectos incluidos
* **Calculadora de sucesiones** para poder calcular sucesiones
* **tablas de multiplicar** para mostrar las tablas de multiplicar
* **calculadora de promedios** esta pensada para calcular un promedio escolar de 3 valores y que la nota minima sea 6
* **calculadora de numeros primos** muestra los numeros primos hasta el valor que ingreses
* **calculadora de equivalencias** muestra equivalencias de medidas en el sistema decimal

## Tecnoloías utilizadas
* **Lenguaje:** Python 3
* **Entorno:** Arch Linux / VSCode / Neovim
* **Control de versiones:** Git / GitHub

## Cómo ejecutarlo
Para correr este programa, asegurate de tener Python instalado y ejecuta:
'''bash
python calculadoras.py
'''
